# UnityBalls

**Summary**

 - Unity Engine C#
 - 2D Casual Game
 - Unfinished


**Features**

 - Drag to shoot controls
 - Random generation of game objects
 - Collision between game objects
 - Bounce calculation
 - Use of the LineRenderer
